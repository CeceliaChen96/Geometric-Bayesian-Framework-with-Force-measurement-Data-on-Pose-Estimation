# Geometric-Bayesian-Framework-with-Force-measurement-Data-on-SO-3-X-R-3
We propose a geometric Bayesian framework which establishes a local exponential-family closure property of the Matrix-- Fisher Gaussian distribution under second-order Bayesian inference. This result establishes a structure-preserving second-order Bayesian update on $\mathrm{SO}(3)\times\mathbb{R}^3$.
