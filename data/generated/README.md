# Generated Data

This folder contains copied generated records used by the paper experiments.

## Contents

- `posterior_estimation/`: JSON, NPZ, and NPY records from the synthetic MFG
  posterior experiments.
- `baseline_experiments/`: JSON and CSV records from the baseline comparison
  scripts.
- `diagnostics/`: JSON and CSV records from diagnostic scripts.

Large generated figures are kept in the original `notebooks/` and `results/`
locations to avoid duplicating plot artifacts.

## Provenance

These files were copied from:

- `notebooks/Posterior Estimation/`
- `notebooks/baseline_experiments/results/`
- `notebooks/diagnostics/`

They are included as fixed generated outputs for repository readers. The
notebooks and scripts remain the authoritative source for how these outputs were
created.
