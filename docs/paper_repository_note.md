# Repository Link Note for Manuscript Review

This note is for author review only. It has not been inserted into the
manuscript.

## Policy Check

IEEE encourages authors to share data and code to improve reproducibility.
IEEE Author Center says reproducibility can be improved by sharing data in an
online repository and uploading code to an online code repository. The IEEE
Control Systems Society author information page for IEEE Transactions on
Automatic Control also notes that Code Ocean and IEEE DataPort are available to
TAC authors.

Based on these sources, a GitHub repository link is appropriate as a
reproducibility resource. I did not find a TAC rule requiring the link to be
placed in the main text specifically. Reasonable options are:

- add a short "Code and data availability" statement near the end of the paper;
- place the statement in the supplementary material;
- provide the repository link through the submission system if TAC requests it.

## Draft Statement

Draft for author approval:

Code and generated data supporting the synthetic experiments and baseline
