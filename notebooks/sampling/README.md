# Sampling Notebook

This directory contains the Matrix-Fisher sampling and visualization notebook:

- `Sampling.ipynb`

The notebook is kept as a provenance/development artifact for the sampling
visualizations used during the project. Generated sampling outputs are not kept
in this folder for the GitHub-facing repository.

For a Python entry point, use:

```bash
python scripts/run_sampling_demo.py
```

If local runs create a `results/` directory under `notebooks/sampling/`, treat
it as disposable generated output rather than source data.
